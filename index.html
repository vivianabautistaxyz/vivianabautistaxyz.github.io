¿Qué es XYZ?
"Somos co-creadores del universo.
Tenemos la capacidad de habitar
el mundo que queremos."

XYZ no es un portafolio ni una empresa. Es un sistema de referencia — el espacio desde el cual observo, interpreto y construyo. Como el plano cartesiano necesita ejes para orientarse, yo necesito coordenadas para leer el mundo. Estas son las mías.

Un sistema en construcción permanente.
No una respuesta — sino el espacio donde habitan las preguntas.

Las manifestaciones

Donde se tocan

Las dimensiones no están separadas. Se intersectan, se nutren, se preguntan mutuamente. XYZ existe precisamente en esos cruces.
